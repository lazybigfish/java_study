## 标识符

1.什么是标识符？

-  java源程序中凡是程序员有权利自己命名的单词都是标识符。
- 有什么是标识符？
- - 类名
  - 方法名
  - 接口名
  - 常量名

2.命名规则，规范。

- 一个合法的标识符只能有‘数字，字母，下划线，美元符号’组成。

- 不能数字开头
- 严格区分大小写
- 关键字不能做标识符
- 要见名知意
- 遵守驼峰命名方式
- - SystemServer
  - CustomServer
- 类名，接口名，首字母大写！后面单词首字母大写
- 变量名，方法名：首字母小写！后面单词首字母大写
- 常量名全部大写！

## 变量

1、什么是变量？

- 变量本质上是内存中的一块空间，这块空间有“数据类型”“名字”“”有字面值
- 变量包含三部分：数据类型、名称、数据
- 

2、数据类型有什么用？

- 不同的数据有不同的数据类型，不同的数据类型底层会分配不同的大小、
- 数据类型是知道程序在运行阶段应该分配多大的空间。

3、变量要求：变量中存储的数据必须与数据类型一致，否则编译报错。

4、声明\定义变量的语法格式：

​	规范要求，首字母小写，后面单词首字母大写。

​	数据类型 变量名 

   例如：int a;

## 数据类型

1、数据类型的作用？

程序中很多数据，每一个数据 都有相关类型，不同的数据类型的数据占用空间大小不同。

数据类型就是知道JVM在程序运行时给该数据分配多大的内存空间。

2、Java重大数据包括：

* 基本数据类型
* 引用

3、关于基本数据类型

- 整数型

- - byte ,short,int,long    /// 1、2/4/8

- 浮点型

- - float,double    ///4/8

- 布尔型

- - boolean   1

- 字符型

- - char   2

## 数组

int[] a; 声明了一个数组变量。
[]表示该变量是一个数组
int 表示数组里的每一个元素都是一个整数
a 是变量名
但是，仅仅是这一句**声明，不会创建数组**

有时候也会写成int a[]; 没有任何区别，就是你看哪种顺眼的问题

```java
public class HelloWorld {
	public static void main(String[] args) {
		// 声明一个数组
		int[] a;
	}
}

```

创建数组时候指定数组长度
